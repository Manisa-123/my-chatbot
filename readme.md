## 🧪 Local Development

1. **Clone the repo**  
   ```bash
   git clone https://github.com/Manisa-123/my-chatbot.git
   cd chatbot-project
   

2. **Create virtual environment**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate

3. **Install dependencies**
    ```bash
   pip install -r requirements.txt

4. **Run the chatbot**
    ```bash
   python app.py
